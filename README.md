## Deploying

Install the [Heroku Toolbelt](https://toolbelt.heroku.com/).

```sh
$ git clone git@github.com:mistah-org/steemph-manila-curator.git # or clone your own fork
$ cd steemph-manila-curator
$ heroku create
$ git push heroku master
$ heroku open
```

or
